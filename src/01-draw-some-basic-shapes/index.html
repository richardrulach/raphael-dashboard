01. Draw some basic shapes

  • You must set up the Raphael drawing object (a paper) by specifying the html element it resides in and the size of the raphael drawing object
  • The raphael paper is a cartesian grid with the origin (0,0) being the top left corner of the paper element (in this case the div)
  • A circle's origin is the centre of the circle.
  • A rectangle's origin is it's top left point
